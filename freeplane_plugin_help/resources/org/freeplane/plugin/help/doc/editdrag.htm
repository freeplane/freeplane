Editing by Drag and Drop

You can move nodes around by using Drag and Drop.

Drag and Drop lets you drop nodes...

...as a child

  • Position the cursor so the destination node is highlighted  to the right.

...as a sibling

  • Position the cursor so the destination node is highlighted  on top.

Hold control while dragging to copy nodes instead of moving them, or drag with the middle mouse button held down.

Before Version 0.7:

If you hold control and shift while dragging, the color and font of the target node is copied to the source node.

You can achieve the same by dragging with the right mouse, target node to source node!! This is contraintuitive!! But it lets you change the color of multiple nodes in one Drag and Drop

If you've selected multiple nodes, they will all be moved or copied.

You can drop sources from external applications

  • Files
    • (on Windows)
  • Pieces of text
    • (works with dropping of text selection in MS Internet Explorer)

You can edit a map by dragging and dropping its file into the background of Freeplane.
