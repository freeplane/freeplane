Keyboard reference

Default keyboard shortcuts. These may be edited in the user.properties file.

 

Menu accelerators:
 new map = Ctrl+N
 open = Ctrl+O
 save = Ctrl+S
 save as = Ctrl+A
 print = Ctrl+P
 close = Ctrl+W
 quit = Ctrl+Q
 export to HTML = Ctrl+E
 export branch to HTML = Ctrl+H
 open first in history = Ctrl+Shift+W
 

Map and mode switching:
 previous map = Ctrl+LEFT
 next map = Ctrl+RIGHT

 mode MindMap = Alt+1
 mode Browse = Alt+2
 mode File = Alt+3

Node editing commands:
 toggle italic = Ctrl+I
 toggle boldface = Ctrl+B
 toggle cloud = Ctrl+shift B

 cut = Ctrl+X
 copy = Ctrl+C
 copy single = Ctrl+Y
 paste = Ctrl+V

 edit = F2
 edit long node = Alt+Enter
 join nodes = Ctrl+J
 toggle folded = SPACE
 toggle children folded = Ctrl+SPACE
 set link by filechooser = Ctrl+shift K
 set link by textfield = Ctrl+K
 set image by filechooser = Alt+K
 node up = Ctrl+UP
 node down = Ctrl+DOWN
 node increase font size = Ctrl+L
 node decrease font size = Ctrl+M
 branch increase font size = Ctrl+Shift+L
 branch decrease font size = Ctrl+Shift+M
 export branch = Alt+A

 node color = Alt+C
 node color blend = Alt+B
 edge color = Alt+E

 

  

Node navigation commands:
 move to root = ESCAPE
 move up = E
 move down = D
 move left = S
 move right = F
 follow link = Ctrl+Enter

New node commands:
 add = Enter
 add child = INSERT
 add sibling before =Shift+Enter

 find = Ctrl+F
 find next = Ctrl+G
