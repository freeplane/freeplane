Installation

Download

  1. Download the Java Runtime Environment.  You need at least J2RE1.4!
  2. Download the Application

Installation

  1. Install Java.
  2. Unpack Freeplane.
  3. On Windows, run Freeplane.exe; you'll probably want to create a folder called freeplane in "c:\Program Files\" and copy the contents of unpacked directory there. If you want, create shortcut of Freeplane.exe on your desktop.
  4. On Unix, run freeplane.sh.

    On both Windows and Mac OS X, you can also simply double click the file freeplane.jar located at the folder lib to execute the program.
