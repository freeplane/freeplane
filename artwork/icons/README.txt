REGEX-BASED LINK DECORATION
Copy linkDecoration.ini into your Freeplane <USER_DIRECTORY> folder. 

To open the Freeplane <USER_DIRECTORY> folder, in Freeplane, 
select Tools > Open user directory

NEW/REPLACEMENT ICONS
To install icons, copy the contents of replacementIcons/ into 
<USER_DIRECTORY>/icons

Be sure to back up your existing icons if you are partial to them.
