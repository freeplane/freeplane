Copying and pasting  

You can copy and paste (multiple) nodes between mindmaps, as you might expect.

You can copy a single node without its children by Copy Single

  • Ctrl + Y

Pasting external data into Freeplane

In addition, you can paste normal text or HTML from other applications.

Normal text

  • Multiple lines are pasted as multiple nodes.
  • URLs are recognized automatically and links are created.
  • Lines with different depth are pasted in different depth.
    e.g.
  • Source
    • Tree
    •     Oak
    •     Beech

As RTF
